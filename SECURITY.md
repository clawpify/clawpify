# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately rather than opening a public issue.

**How to report:**

- Open a [GitHub Security Advisory](https://github.com/YOUR_ORG/clawpify/security/advisories/new) (preferred), or
- Email the maintainers directly

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond as soon as possible and keep you updated on the status of the fix.
